### LuisCountDownTimer  
Android 倒计时控件，支持倒计时，一个java文件支持倒计时。

#### 概述  
````
网上多数都是采用handler来实现倒计时，感觉比较复杂，自己采用了Android自带的CountDownTimer来简单实现了倒计时（虽然底层还是使用的handler），这里仅提供了一个简单的实现思路给大家参考下。
````
